# E Commerce Site:-

## Database Modeling:-

### Users:-

- [x] userName -> String
- [x] firstName -> String
- [x] lastName -> String
- [x] email -> String
- [x] password -> String
- [x] profileImg -> String

---

### Products:-

- [x] name -> String
- [x] desc -> String
- [x] count -> Int64
- [x] image -> Array -> String
- [x] review -> Array -> Object -> {name, email, image, review, createdAt}
- [x] price -> String
- [x] createdAt -> Date().now()

---

### Orders:-

- [x] userId -> String
- [x] userName -> String
- [x] productImages -> Array -> Object -> {itemName, itemImage, unitPrice, counts}
- [x] totalCost -> Int64
- [x] status -> Array -> Object -> {orderPlaced, inProgress, outForDelivery, delivered} => Boolean
